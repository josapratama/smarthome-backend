Quick API Reference

Essential endpoints to get started with the Smart Home Backend API.

Authentication

POST /api/v1/auth/login Login
POST /api/v1/auth/register Register
POST /api/v1/auth/refresh Refresh Token

Home Management

GET /api/v1/homes List Homes
POST /api/v1/homes Create Home
GET /api/v1/homes/{id}/devices Home Devices

Device Control

POST /api/v1/devices/{id}/telemetry Send Data
POST /api/v1/devices/{id}/commands Send Command
GET /api/v1/devices/{id}/telemetry Get History

AI & Analytics

POST /api/v1/devices/{id}/energy-predictions Predict Energy
GET /api/v1/devices/{id}/ai-insights AI Insights
POST /api/v1/predictions/{id}/anomalies Detect Anomalies

Alarms & Notifications

GET /api/v1/homes/{id}/alarms List Alarms
POST /api/v1/homes/{id}/alarms Create Alarm
GET /api/v1/notifications/endpoints Notification Endpoints

Firmware & OTA

GET /api/v1/firmware/releases List Firmware
POST /api/v1/devices/{id}/ota Trigger OTA
GET /api/v1/ota/jobs/{id} OTA Status

Real-time IoT

MQTT-based real-time communication with IoT devices, supporting ESP32, Arduino, and custom hardware.

AI Analytics

Machine learning-powered energy prediction, anomaly detection, and intelligent insights for smart home optimization.

Enterprise Security

JWT authentication, role-based access control, device authentication, and comprehensive audit logging.

Scalable Database

PostgreSQL with optimized indexing, soft deletes, and efficient queries for handling millions of sensor readings.

OTA Updates

Over-the-air firmware updates with version management, rollback support, and batch deployment capabilities.

Smart Alarms

Intelligent alarm system with customizable thresholds, multi-channel notifications, and automated escalation.
